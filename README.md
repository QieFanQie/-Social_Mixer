这是一个获取社交平台推送然后扒到一起推送的框架；其中b站用的是一种方法，贴吧lofter小红书用的是另一种。
绝大部分代码是AI写的。
我自己测试用的是egde浏览器，其他的会出问题也说不定呢。
使用方法：
  下载并在浏览器插件页面选择加载本地的插件，选择文件夹就可以了。
  加载之前最好登录上各平台账号。
  加载之后，新建文件夹会被我替换为聚平台。按理来说打开网页之后会进行一次自动的信息获取（网页右上角会显示有什么），如果遇到显示0个帖子0个作品等的情况，手动点刷新全部并在刷新时的打开网页过程中不要进行其他操作应该能解决问题。
其中，b站是获取你的推送；贴吧是获取主页内容；lofter是获取发现页内容；小红书就是点进去的页面。
主要就是展示一下这个点子。
